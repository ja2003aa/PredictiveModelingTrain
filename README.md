# PredictiveModeling

# Installation/setup
- make a venv
- run pip install -r requirements.txt
- install firefox
- install geckodriver v0.34.0 from https://github.com/mozilla/geckodriver/releases

# Run 
Scraper - Python3 scraper_demo.py (tested with Python 3.8.10)
Chatbot - streamlit run \UI_Chatbot.py
